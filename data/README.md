# Data Folder

This folder contains the datasets used in the **ML-Based Network Intrusion Detection System (NIDS) with Real-Time Alerting** project.

## Folder Structure

* `raw/` → stores the original dataset files
* `processed/` → stores cleaned or transformed datasets (if applicable)

## Current Dataset Files

* `raw/KDDTest+.csv`
* `raw/network.csv`

## Notes

* Large dataset files may be excluded from the GitHub repository to keep it clean and within file size limits.
* If a dataset is missing, place it inside the `data/raw/` folder before running the project.
* If preprocessing is used, the cleaned output should be saved in `data/processed/`.

## Example Paths

* `data/raw/KDDTest+.csv`
* `data/raw/network.csv`
