# ML-Based Network Intrusion Detection System (NIDS)

A Machine Learning-based Network Intrusion Detection System with real-time Nmap scan detection and CSV-based traffic analysis. Built using Streamlit and Scikit-learn.

---

## Features

- Real-time Nmap scan detection using packet sniffing (Scapy)
- Machine learning intrusion detection (Random Forest + Isolation Forest)
- CSV upload for offline traffic analysis
- Interactive dashboard with charts and metrics
